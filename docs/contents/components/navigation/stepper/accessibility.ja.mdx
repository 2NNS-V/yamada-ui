## ARIAロールと属性

| コンポーネント  | ロールと属性         | 使い方                                       |
| --------------- | -------------------- | -------------------------------------------- |
| `StepSeparator` | `role='separator'`   | セパレータであることを示します。             |
| `StepStatus`    | `aria-hidden='true'` | 要素をアクセシビリティツリーから除外します。 |
