## ARIA Roles and Attributes

| Component    | Role and Attributes                | Usage                                                     |
| ------------ | ---------------------------------- | --------------------------------------------------------- |
| `Pagination` | `role="navigation"`                | Indicates that this is a navigation region between pages. |
|              | `aria-label="Go to first page"`    | Indicates a button for navigating to the first page.      |
|              | `aria-label="Go to previous page"` | Indicates a button for navigating to the previous page.   |
|              | `aria-label="Go to next page"`     | Indicates a button for navigating to the next page.       |
|              | `aria-label="Go to last page"`     | Indicates a button for navigating to the last page.       |
