## ARIAロールと属性

| コンポーネント                   | ロールと属性        | 使い方                                                                                                                                                                  |
| -------------------------------- | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Pagination`                     | `role="navigation"` | ページ間のナビゲーション領域であることを示します。                                                                                                                      |
| `PaginationItem` <Tag>内部</Tag> | `aria-label`        | `"Go to first page"`, `"Go to previous page"`, `"Go to next page"`, `"Go to last page"`が設定されます。`page`が設定されている場合は`"Go to page ${page}"`を設定します。 |
