## ARIAロールと属性

| コンポーネント | ロールと属性                       | 使い方                                                     |
| -------------- | ---------------------------------- | ---------------------------------------------------------- |
| `Pagination`   | `role="navigation"`                | ページ間のナビゲーション領域であることを示します。         |
|                | `aria-label="Go to first page"`    | 最初のページに移動するためのボタンであることを示します。   |
|                | `aria-label="Go to previous page"` | 一つ前のページに移動するためのボタンであることを示します。 |
|                | `aria-label="Go to next page"`     | 一つ後のページに移動するためのボタンであることを示します。 |
|                | `aria-label="Go to last page"`     | 最後のページに移動するためのボタンであることを示します。   |
