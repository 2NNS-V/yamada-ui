## ARIAロールと属性

| コンポーネント | ロールと属性        | 使い方                                                                                                                                        |
| -------------- | ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| `Pagination`   | `role="navigation"` | ページ間のナビゲーション領域であることを示します。                                                                                            |
|                | `aria-label`        | `page`に`"ellipsis"`が設定されている場合は`"Jump to omitted pages"`を設定し、`page`が設定されている場合は`"Go to page ${page}"`を設定します。 |
