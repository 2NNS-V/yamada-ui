## Keyboard Navigation

| Key          | Description                                                       | State |
| ------------ | ----------------------------------------------------------------- | ----- |
| `ArrowDown`  | Decreases the slider value by one step.                           | -     |
| `ArrowLeft`  | Decreases the slider value by one step.                           | -     |
| `ArrowRight` | Increases the slider value by one step.                           | -     |
| `ArrowUp`    | Increases the slider value by one step.                           | -     |
| `End`        | Moves to the maximum value of the slider.                         | -     |
| `Home`       | Moves to the minimum value of the slider.                         | -     |
| `PageDown`   | Significantly decreases the slider value. (by 10% of total range) | -     |
| `PageUp`     | Significantly increases the slider value. (by 10% of total range) | -     |

## ARIA Roles and Attributes

| Component   | Role and Attributes         | Usage                                                                                  |
| ----------- | --------------------------- | -------------------------------------------------------------------------------------- |
| SliderThumb | `role="slider"`             | Indicates that the element is a slider.                                                |
|             | `aria-label="Slider thumb"` | Sets the description for the slider's thumb.                                           |
|             | `aria-valuemax`             | Sets the maximum value of the slider. If `max` is not set, defaults to `360`.          |
|             | `aria-valuemin`             | Sets the minimum value of the slider. If `min` is not set, defaults to `0`.            |
|             | `aria-valuenow`             | Sets the current value of the slider. If `defaultValue` is not set, defaults to `180`. |
|             | `aria-disabled`             | Set to `"true"` when `isDisabled` is set on the `HueSlider`.                           |
|             | `aria-readonly`             | Set to `"true"` when `isReadOnly` is set on the `HueSlider`.                           |
