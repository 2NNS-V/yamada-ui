The `Slider` follows the [WAI-ARIA - Slider Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/slider/) for accessibility.

## Keyboard Navigation

| Key          | Description                                                                            | State |
| ------------ | -------------------------------------------------------------------------------------- | ----- |
| `ArrowRight` | Increase the value of the slider by one step.                                          | -     |
| `ArrowUp`    | Increase the value of the slider by one step.                                          | -     |
| `ArrowLeft`  | Decrease the value of the slider by one step.                                          | -     |
| `ArrowDown`  | Decrease the value of the slider by one step.                                          | -     |
| `Home`       | Set the slider to the first allowed value in its range.                                | -     |
| `End`        | Set the slider to the last allowed value in its range.                                 | -     |
| `PageUp`     | Increase the slider value by an amount larger than the step change made by Up Arrow.   | -     |
| `PageDown`   | Decrease the slider value by an amount larger than the step change made by Down Arrow. | -     |

## ARIA Roles and Attributes

| Component   | Role and Attributes         | Usage                                                                                  |
| ----------- | --------------------------- | -------------------------------------------------------------------------------------- |
| SliderThumb | `role="slider"`             | Indicates that the element is a slider.                                                |
|             | `aria-label="Slider thumb"` | Sets the description for the slider's thumb.                                           |
|             | `aria-valuemax`             | Sets the maximum value of the slider. If `max` is not set, defaults to `360`.          |
|             | `aria-valuemin`             | Sets the minimum value of the slider. If `min` is not set, defaults to `0`.            |
|             | `aria-valuenow`             | Sets the current value of the slider. If `defaultValue` is not set, defaults to `180`. |
|             | `aria-disabled`             | Set to `"true"` when `isDisabled` is set on the `HueSlider`.                           |
|             | `aria-readonly`             | Set to `"true"` when `isReadOnly` is set on the `HueSlider`.                           |
