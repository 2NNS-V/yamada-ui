## キーボード操作

| キー         | 説明                                                                   | 状態 |
| ------------ | ---------------------------------------------------------------------- | ---- |
| `ArrowRight` | スライダーの値を1段階増加させます。                                    | -    |
| `ArrowUp`    | スライダーの値を1段階増加させます。                                    | -    |
| `ArrowLeft`  | スライダーの値を1段階減少させます。                                    | -    |
| `ArrowDown`  | スライダーの値を1段階減少させます。                                    | -    |
| `Home`       | スライダーを範囲内の最初の許容値に設定します。                         | -    |
| `End`        | スライダーを範囲内の最後の許容値に設定します。                         | -    |
| `PageUp`     | 上矢印による段階的な変更よりも大きな量でスライダーの値を増加させます。 | -    |
| `PageDown`   | 下矢印による段階的な変更よりも大きな量でスライダーの値を減少させます。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性                | 使い方                                                                                                        |
| -------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------- |
| SliderThumb    | `role="slider"`             | スライダーであることを示します。                                                                              |
|                | `aria-label="Slider thumb"` | スライダーの`thumb`の説明を設定します。                                                                       |
|                | `aria-valuemax`             | スライダーの最大値で`max`の値を設定します。`max`が設定されていない場合は`360`を設定します。                   |
|                | `aria-valuemin`             | スライダーの最小値で`min`の値を設定します。`min`が設定されていない場合は`0`を設定します。                     |
|                | `aria-valuenow`             | スライダーの現在値で`defaultValue`の値を設定します。`defaultValue`が設定されていない場合は`180`を設定します。 |
|                | `aria-disabled`             | `"HueSlider"`に`isDisabled`が設定されている場合は`"true"`を設定します。                                       |
|                | `aria-readonly`             | `"HueSlider"`に`isReadOnly`が設定されている場合は`"true"`を設定します。                                       |
