## キーボード操作

| キー    | 説明                                   | 状態 |
| ------- | -------------------------------------- | ---- |
| `Tab`   | トグルをフォーカスします。             | -    |
| `Enter` | フォーカスされたトグルを切り替えます。 | -    |
| `Space` | フォーカスされたトグルを切り替えます。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性   | 使い方                                                                |
| -------------- | -------------- | --------------------------------------------------------------------- |
| `ToggleGroup`  | `role="group"` | グループであることを示します。                                        |
| `Toggle`       | `aria-pressed` | トグルがオンの場合は`true`を設定し、オフの場合は`false`を設定します。 |
