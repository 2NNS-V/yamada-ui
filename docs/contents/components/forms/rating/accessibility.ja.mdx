## キーボード操作

| キー         | 説明                               | 状態 |
| ------------ | ---------------------------------- | ---- |
| `ArrowLeft`  | 前の`RatingItem`の選択を外します。 | -    |
| `ArrowRight` | 次の`RatingItem`を選択します。 　  | -    |
| `ArrowUp`    | 前の`RatingItem`の選択を外します。 | -    |
| `ArrowDown`  | 次の`RatingItem`を選択します。     | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性         | 使い方                                                   |
| :------------- | :------------------- | :------------------------------------------------------- |
| `Rating`       | `aria-readonly`      | `isReadOnly`が設定されている場合は`"true"`を設定します。 |
| `RatingItem`   | `aria-label`         | `value`を設定します。                                    |
|                | `aria-hidden="true"` | `RatingIcon`をアクセシビリティツリーから除外します。     |
