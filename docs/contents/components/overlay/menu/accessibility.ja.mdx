`Menu`は、アクセシビリティに関して[WAI-ARIA - Menu and Menubar Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menubar/)および[WAI-ARIA - Menu Button Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menu-button/)に従います。

## キーボード操作

| キー         | 説明                                                                                                                                                                                                                                                                                                                                                 | 状態 |
| ------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| `Space`      | ボタンにフォーカスがある場合、メニューを開き、最初の項目にフォーカスを置きます。チェックボックスもしくはラジオボタンにフォーカスがある場合、メニューを閉じずに状態を更新します。サブメニューを持つ項目にフォーカスがある場合、サブメニューが開きその最初の項目にフォーカスが置かれます。それ以外の場合は、項目をアクティブにしてメニューを閉じます。 | -    |
| `Enter`      | ボタンにフォーカスがある場合、メニューを開き、最初の項目にフォーカスを置きます。サブメニューを持つ項目にフォーカスがある場合、サブメニューが開きその最初の項目にフォーカスが置かれます。それ以外の場合は、項目をアクティブにしてメニューを閉じます。                                                                                                 | -    |
| `ArrowRight` | サブメニューを持つ項目にフォーカスがある場合、サブメニューが開きその最初の項目にフォーカスが置かれます。                                                                                                                                                                                                                                             | -    |
| `ArrowLeft`  | サブメニューにフォーカスがある場合、サブメニューを閉じて親にフォーカスを戻します。                                                                                                                                                                                                                                                                   | -    |
| `ArrowUp`    | ボタンにフォーカスがある場合、メニューを開き、最後の項目にフォーカスを置きます。フォーカスがメニュー内にある場合、フォーカスを前の項目に移動します。最初の項目にフォーカスがある場合は、最後の項目にフォーカスを移動します。                                                                                                                         | -    |
| `ArrowDown`  | ボタンにフォーカスがある場合、メニューを開き、最初の項目にフォーカスを置きます。フォーカスがメニュー内にある場合、フォーカスを次の項目に移動します。最後の項目にフォーカスがある場合は、最初の項目にフォーカスを移動します。                                                                                                                         | -    |
| `Home`       | 現在のメニューの最初の項目にフォーカスを移動します。                                                                                                                                                                                                                                                                                                 | -    |
| `End`        | 現在のメニューの最後の項目にフォーカスを移動します。                                                                                                                                                                                                                                                                                                 | -    |
| `Escape`     | フォーカスを含むメニューを閉じ、親要素にフォーカスをもとします。                                                                                                                                                                                                                                                                                     | -    |

## ARIAロールと属性

| コンポーネント    | ロールと属性            | 使い方                                                                                                                                                            |
| ----------------- | ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MenuButton`      | `role="button"`         | `Button`であることを示します。                                                                                                                                    |
|                   | `id`                    | メニューと関連付けるために使用する`id`。                                                                                                                          |
|                   | `aria-haspopup`         | `menu`を設定します。                                                                                                                                              |
|                   | `aria-expanded`         | メニューが表示されている場合、`"true"`を設定します。メニューが非表示の場合は、`"false"`を設定します。                                                             |
|                   | `aria-controls`         | 関連したメニューの`id`を設定します。                                                                                                                              |
| `MenuList`        | `role="menu"`           | メニューであることを示します。                                                                                                                                    |
|                   | `tabindex`              | `"-1"`を設定します。                                                                                                                                              |
|                   | `aria-activedescendant` | フォーカスされた項目の`id`を設定します。                                                                                                                          |
|                   | `aria-labelledby`       | 表示を制御するメニュー項目またはボタンを参照する`id`に設定されます。                                                                                              |
| `MenuItem`        | `tabindex`              | フォーカスされた場合、`"0"`を設定します。そうでなければ`"-1"`を設定します。                                                                                       |
|                   | `id`                    | メニューと関連付けるために使用する`id`。                                                                                                                          |
|                   | `aria-disabled`         | 項目が無効な場合、`"true"`を設定します。                                                                                                                          |
| `MenuDivider`     | `role="separator"`      | セパレーターであることを示します。                                                                                                                                |
|                   | `aria-orientation`      | `"horizontal"`を設定します。                                                                                                                                      |
| `MenuOptionGroup` | `role="group"`          | グループであることを示します。                                                                                                                                    |
| `MenuOptionItem`  | `role`                  | `MenuOptionGroup`の`type`が`"radio"`に設定されている場合は、`"menuitemradio"`を設定します。`"checkbox"`に設定されている場合は、`"menuitemcheckbox"`を設定します。 |
|                   | `tabindex`              | フォーカスされた場合、`"0"`を設定します。そうでなければ`"-1"`を設定します。                                                                                       |
|                   | `aria-checked`          | チェックされている場合は`"true"`を設定し、チェックされていない場合は`"false"`を設定します。                                                                       |
|                   | `aria-disabled`         | 項目が無効な場合、`"true"`を設定します。                                                                                                                          |
