`Table`は、アクセシビリティに関して[WAI-ARIA - Table Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/table/)に従います。

`Table`に`aria-label`を設定すると、スクリーンリーダーによって読み上げられます。

```tsx live=false
<Table columns={columns} data={data} aria-label="DRAGON BALL characters" />
```

## キーボード操作

| キー            | 説明                                                                                                             | 状態 |
| --------------- | ---------------------------------------------------------------------------------------------------------------- | ---- |
| `Tab`           | フォーカスがテーブルに移動すると最初の項目をフォーカスします。テーブル内である場合は次の項目をフォーカスします。 | -    |
| `Shift` + `Tab` | 無効ではない前の項目をフォーカスします。                                                                         | -    |
| `ArrowLeft`     | 無効ではない前の項目をフォーカスします。                                                                         | -    |
| `ArrowRight`    | 無効ではない次の項目をフォーカスします。                                                                         | -    |
| `ArrowUp`       | 無効ではない前の項目をフォーカスします。                                                                         | -    |
| `ArrowDown`     | 無効ではない次の項目をフォーカスします。                                                                         | -    |
| `Home`          | 無効ではない最初の項目をフォーカスします。                                                                       | -    |
| `End`           | 無効ではない最後の項目をフォーカスします。                                                                       | -    |

## ARIAロールと属性

| コンポーネント       | ロールと属性    | 使い方                                                                                                          |
| -------------------- | --------------- | --------------------------------------------------------------------------------------------------------------- |
| `Table`              | `role="table"`  | テーブルであることを示します。                                                                                  |
|                      | `aria-rowcount` | テーブルの行数を示します。                                                                                      |
|                      | `aria-colcount` | テーブルの列数を示します。                                                                                      |
| `Tr` <Tag>内部</Tag> | `aria-disabled` | 行の`id`が`disabledRowIds`に含まれている場合は、`"true"`を設定します。                                          |
|                      | `aria-selected` | 行が選択済みの場合は、`"true"`を設定し、未選択の場合は`"false"`を設定します。                                   |
| `Th` <Tag>内部</Tag> | `aria-colindex` | 列がテーブルの何列目であるかを示します。                                                                        |
|                      | `aria-rowindex` | 行がテーブルの何行目であるかを示します。                                                                        |
|                      | `aria-sort`     | 列が昇順の場合は`"ascending"`を設定し、降順の場合は`"descending"`を設定、指定がない場合は`"none"`を設定します。 |
